Informacion General OVI

Es un recurso digital diseñado para brindar información a los usuarios e interactuar con ellos. Es a partir de estos que podemos entregar información a través de: Páginas Web, Blog, Correo Electrónico, Redes Sociales, etc.

Objetivo General

Este OVI (Objeto Virtual de Información), está orientado a la planificación para diseñar un Sitio Web, el cual pretende dirigir los pasos en la vinculación de una base de datos en una página desde PHP con opciones de consulta, edición, modificación y eliminación de datos y/o registros.

Objetivos Específicos

    Mostrar la programación y los elementos necesarios para la vinculación de BD a una página con PHP.
    Planear, implementar y evaluar Diseño de Sitios Web para su aplicación.
    Crear una maquetación o Mockup de la página Index del sitio web.
